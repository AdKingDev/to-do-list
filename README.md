# To Do List

Uma aplicação de lista de tarefas desenvolvida com foco em organização, usabilidade e animações modernas. O projeto foi criado com o objetivo de praticar JavaScript, manipulação do DOM e armazenamento local utilizando LocalStorage.

Funcionalidades:
- Adicionar tarefas
- Remover tarefas
- Validação de tarefas duplicadas
- Armazenamento no LocalStorage
- Animações com ScrollReveal
- Interface responsiva

Link do projeto: https://to-do-list-pink-six-54.vercel.app/
